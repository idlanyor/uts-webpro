## Installation

clone my repository

``` bash
    git clone https://github.com/roynaldi3301/uts-webpro
```
run installation
  ```bash
  npm run dev
  ```
run vite server in dev mode
```
npm run dev
```
linting
```
npm run lint
```
build vite
```
npm run build
```
